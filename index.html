Important Concepts

Lesson 1: The Basics of the Web and HTML

How the Web Works

The web is a network of computers that communicate via the internet. When someone types in a web address their computer sends a HTTP Request to a server.

The server finds the corresponding HTML document and sends it back to the computer where a browser interprets the code and displays it as the file requested.

Main components of the web include HTML (the main document type), URLs (web addresses), and HTTP (the main protocol the web runs on).

HTML

HTML is the acronym for Hypertext Markup Language. HTML documents form most of the content on the web.

HTML documents contain text which describes "what you see" and markup which describes "how it looks".

Tags and Elements

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags.

Computers are Stupid

Computers interpret instructions literally. This makes it very easy for a small error to turn in to a major issue for the progranner.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them adding height and width.

Lesson 2: Creating a Structured Document with HTML

The "tree-like structure" of HTML

The "tree-like structure" means that HTML elements can have other elements inside of them. You can think of this relationship like a family tree. In a browser, this structure shows up as a series of nested boxes. There are boxes inside of boxes inside of boxes, etc.

The relationship between indented HTML and boxes

When an HTML document is read as text, you see a wave of indentations up and down the left side of the document. The more indented an element is, the more deeply nested it is.

Text Editors (for programming)

When writing code, programmers can use special text editors that make their lives easier. Some text editors add the closing tag when an opening is typed and some suggest tags as you type.

Lesson 3: Adding CSS Style to HTML Structure

Avoiding Repetition

Avoiding repetition is very important. It makes editing easier and reduces errors by limiting the number of changes that have to be manually made to lines of code.

By assigning the same class to divs throughout the document we can assign the same styles to similar parts without having to markup each part individually and risk making a mistake.
CSS

CSS stands for Cascading Style Sheets. This allows programmers to control the style of related HTML elements. This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class.

Telling Computers What to Do

Lesson 1: Introduction to "Serious" Programming

Computer

Most machines are made to do a few things. Unless you physically change these machines, they will only be able to do those things.

Computers are different in that they can be programmed to do most anything, as long as we write a program with specific sequences of instructions it can follow.

Computer Program

A program is made up of precise sequences of steps a computer can follow. Programs include things such as games, mobile apps, and even browsers.

Programming Language

Programming languages are used by programmers to tell a computer what to do. Python is one example of a programming language.

Python

When you write Python code and "Run" the code, a Python Interpreter converts the written code into a set of instructions that the computer can understand and execute.

Grammar

Just like in a language such as English, programming languages have grammar. Grammar specifies what is correct and what is incorrect.

While people can sometimes make sense of sentences that may not be technically grammatically correct computers aren't smart enough to do this. Code must be written exactly "correct" according to the Python interpreter, otherwise it will not give us the intended result or will produce an error.

Python Expressions

In Python an "expression" is a legal Python statement. For example: print 2 + 2 is a valid expression, but print 2 + (without a number at the end) is not.

Lesson 2: Variables and Strings

What is a variable in Python?

Variables give programmers a way to give names to values. If my_variable is a variable with a value of 2, then the following code would print out 0:

print my_variable - my_variable

What does it mean to assign a value to a variable? How is it done?

We can assign the value 2 to the variable my_variable by writing code like this:

my_variable = 2

We can change the value of a variable simply by re-assigning it a different value later.

What's the difference between the equal sign in 3 + 3 = 6 and my_variable = 6?

In the line 3 + 3 = 6, the equals sign means "is the same as".

In the line my_variable = 6, the equals sign means "takes the value of".

What are some ways variables are useful?

Variables can be useful to programmers in many ways:

  • They make programming code easier to read by using names that make sense to those reading it.
  • They give us a way to store the value of data.
  • They give us an easy way to change the value of something
What is the difference between 2 + 2 and "2" + "2"?

In Python, 2 is a number while "2" is a string.

The code 2 + 2 would give 4.

The code "2" + "2" would give "22".

Lesson 3: Input -> Function -> Output

What is a function, also known as a "procedure"?

A function is something that takes input, does something to that input, and then produces output. For example, a function named square might take a number as input and produce the square of that number as output.

What is the difference between making and using a function?

Functions are made by starting a line of code with the keyword def (meaning define) and then giving the function a name followed by the function parameters in parentheses. These parameters will be replaced real values when the function is used.

In the "body" of the function, we write the code that specifies what to do with the input. The following code could be the definition of a function called sum:

def sum(x,y):
answer = x + y
return answer

To use a function, we write the name of the function followed by the value(s) we want to give it in parentheses. Like this:

print sum(4,3)
>>>7
How do functions help programmers avoid repetition?

Functions can be created and reused over and over. Once you've defined a function, you never have to define it again.

What happens if a function doesn't have a return statement?

The return keyword tells Python what the function should produce as output. If a function doesn't have a return statement, then it does not know what to output and does nothing.

Using IF and ELSE and OR Statements

An else statement can be combined with an if statement. An else statement contains the block of code that executes if the conditional expression in the if statement resolves to 0 or a false value. An or statement only evaluate what it needs. I.E. if the 1st expression evaluates to true then the 2nd expression is not evaluated. If the 1st expression evaluates to false then the value of the 2nd statement is used.

Loops and Breaks

A while loop statement repeatedly executes a target statement as long as a given condition is true. When the condition becomes false, it passes to the line immediately following the loop.

The break statement in Python terminates the current loop and resumes execution at the next statement.

Debugging

Debugging code when something does not return the intended result can be done by breaking large amounts of code down in to smaller amounts of code and printing the results until we determine where the code "breaks". We can then make the necessary changes and save our program.

Nested Lists

Simply put a 'nested list' can be defined as a list that occurs as an element of another list. Nested lists appear in brackets within brackets separated by commas i.e [1,2,3, [4,5,6]]. In this example 1,2,3 would be the first element or element 0 and 4,5,6 would be the second element or element 1.

Mutations

With strings when we changed the value of a variable in a string we change just the variable but not the string itself. Changing the value of a variable in a list can change the value of the list itself.

Aliasing

Aliasing is when two variables refer to the same object. i.e.
p = ['H', 'e', 'l', 'l', 'o']
p[0] = 'Y'
q = p
After this assignment, p and q refer to the same list: ['Y', 'e', 'l', 'l', 'o']. For each mutation of the referred to object other variables that refer to that object change as well.

Built in Operations

Built in operations such as append to add elements to existing lists, concatenation (+) to add lists to create a new list without changing the input of either of the original, and len to detemine the length of number of elements of its input.
